#Project Overview
This is a post-style application built as part of the Elevate Frontend Advanced Bootcamp. The project focuses on building a pixel-perfect UI and integrating with the JSONPlaceholder API to manage and display posts. It demonstrates clean code practices, scalable architecture, and robust form handling.





‚ú® Key Features

Pixel-Perfect UI: Strictly follows the provided Figma design for a professional look and feel.



API Integration: Full integration with JSONPlaceholder for fetching and managing posts.


Form Validation: Advanced form handling using React Hook Form and Zod to ensure data integrity.



Responsive Design: Fully accessible and responsive across all device sizes.


User Feedback: Integrated Toast messages for real-time notifications on user actions.

üöÄ Getting Started
Prerequisites
Make sure you have Node.js and npm installed on your machine.

Installation
Clone the repository:

Bash

git clone https://github.com/AbdoAmmar96/Elevate-Task.git
Navigate to the project directory:

Bash

cd Elevate-Task
Install dependencies:

Bash

npm install
Running the Project
To start the development server:

Bash

npm run dev
# or
npm start
